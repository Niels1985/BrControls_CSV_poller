# BrControls_CSV_poller
Small program to load in a CSV / Excel file used bij BrControls and partners to generare .cpt files.
This program lets you poll directly with al RS485 (USB => RS485 converter) to the slave device.
It makes it a lot mor easely and quicker to check your .cpt in the controller.
Program has to be the ancer on BrWorkReg who is difficult to run inder modern OS.
